# DataAnalysis_Project_SeriesGlobales

**Projet d’analyse de données** — Utilisation des méthodes vues en cours : ACP (PCA), régression linéaire, ANOVA / éventuellement régression logistique — appliquées au jeu de données *SeriesGlobales*.

## Objectifs

- Lire et comprendre le jeu de données « SeriesGlobales ».  
- Réaliser une **analyse descriptive** (statistiques, visualisations).  
- Appliquer des méthodes statistiques :
  - Analyse en Composantes Principales (ACP / PCA)  
  - Régression linéaire multiple  
  - ANOVA (si applicable)  
  - Régression logistique (si une variable binaire peut être construite)  
- Générer des graphiques et résumer les résultats.  
- Produire un rapport structuré (introduction, méthodes, résultats, discussion).
